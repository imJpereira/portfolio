    Sou um desenvolvedor web, me conheça melhor a partir deste portifólio.

    Sobre mim

    Comecei minha jornada no mundo de desenvolvimento de software em 2021, com um curso de algoritmo, do curso em vídeo, que foi o ponto de partida para aprender sobre lógica de programação e resolução de algoritmos.


    No ano posterior, comecei a estudar o desenvolvimento de páginas web, através de um curso na Udemy. Durante este ano aprendi muito sobre HTML, CSS e Javascript, fazendo este curso, lendo documentações e participando de eventos online, como a imersão front-end da alura e o intensivo de javascript do canal hashtag programação.


    Também adquiri ao longo do tempo um conhecimento básico sobre back-end, aprendendo sobre NodeJS e base de dados. Atualmente estou cursando Ciência da Computação, na Atitus Passo Fundo, visando ampliar ainda mais o meu conhecimento.

    Sou uma pessoa que gosta de se comunicar, acredito que uma boa comunicação pode resolver muitos problemas que parecem mais complexos do que realmente são. Além disso, sou bem aberto para perspectivas diferentes, busco sempre melhorar, então é justo escutar outras opiniões que podem agregar.


    Como mencionei anteriormente, estou focado em melhorar constantemente, o que me leva a passar a maior parte do meu dia estudando. Isso tem me permitido aprender de maneira rápida e eficaz. Para mim, o mais crucial é evitar a estagnação, me mantendo atualizado e sempre buscando desafios.

    Conhecimentos

    HTML

    CSS

    Javascript

    NodeJS

    MySQL

    MongoDB

    Git

    GitHub

    Tenho um bom conhecimento em HTML e consigo criar os elementos de um site com foco no estilo que será aplicado posteriormente. Entendo quando devo utilizar tags específicas para construir uma boa estrutura e semântica em minhas páginas.

    Tenho um conhecimento amplo em CSS, utilizo muitas funcionalidades como: flex-box, grid-box, pseudoclasses, entre outras. Além disso sei deixar um site responsivo utilizando a media queries.

    Tenho conhecimento das funcionalidades do JavaScript, incluindo conceitos como orientação a objetos, webpack, polimorfismo e requisições assíncronas com AJAX (utilizando tanto o fetch quanto o axios).


    Além disso, também sei como manipular elemtos do DOM, seja utilizando Javascript puro, seja utilizando a biblioteca JQuery para simplificar o processo.

    Tenho conhecimento básico em NodeJS e já utilizei o framework Express, onde aprendi a utilizar a arquitetura MVC(Model-View-Controller) para estruturar melhor meu projeto


    Além disso, também sei como manipular rotas e realizar conexões com bancos de dados MongoDB utilizando o mongoose para facilitar a interação com os dados.

    Tenho conhecimento básico em MySQL e estou familiarizado com operações de manipulação de dados, como criar, ler, atualizar e excluir registros (CRUD).


    Compreendo os conceitos fundamentais de bancos de dados relacionais, incluindo o uso de primary keys e foreign keys. Minha experiência foi toda utilizando o MySQL Workbench e o WAMP server.

    Embora meu conhecimento sobre MongoDB seja básico, compreendo o funcionamento das collections e sou capaz de realizar operações de CRUD utilizando NodeJS.

    Tenho um conhecimento amplo em Git, entendendo suas principais funcionalidades. O Git é uma ferramenta fundamental para mim, sendo o meio de versionamento que utilizo em todos os meus projetos.

    O GitHub é o meio em que hospedo todos os meus projetos. Sei como funciona o sistema de repositório remoto, issues e pull requests.
